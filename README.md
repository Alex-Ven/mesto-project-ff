## Функциональность

- Создание и публикация карточек на странице
- Изменение аватара профиля
- Модальные окна
- Валидация форм редактирования профиля, аватара пользователя и добавления новой карточек места.
- Интеграция с API.
- Добавление, удаление и лайк карточек места.
